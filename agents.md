# Some information for agents

## Directories
- ./docs	Output documentation such as installation instructions
- ./questions	Documents containing questions for the operator/developer
- ./research	Information gathered and summary file
- ./assets	Image and sound assets
- ./raw		Raw input files, not part of delivered project, raw materials processed into assets
- ./dist	Distributable files e.g to be published to github pages
;add additional directories here as required

## Style
- Programs include some comments per function
- requirements.txt is always maintained across all python scripts
- python scripts are kept specialised and fairly small
- HTML+Javascript is used to deliver content to gh-pages in one page; compatible with all recent browsers
